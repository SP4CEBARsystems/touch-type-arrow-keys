# Touch Type Arrow Keys
Emulates the arrow keys and other common keys from the touch-typing pose

## Controls
layer activation: hold right shift or toggle menu key

## Cursor Controller
### Arrows
| key | action |
|---|---|
| e | up |
| s | left |
| d | down |
| f | right |

### Cursor Jump
| key | action |
|---|---|
| w | home |
| r | end |
| q | page up |
| a | page down |

### Removal
| key | action |
|---|---|
| t | backspace |
| g | delete |

### Clipboard
| key | action |
|---|---|
| z | undo |
| x | cut |
| c | copy |
| v | paste |

### Mouse Emulation
| key | action |
|---|---|
| j | click left |
| k | middle |
| l | right |
| i | scroll up |
| o | down |
